Sobre

Esse portal de filmes foi desenvolvido como atividade prática final avaliativa para a disciplina de Desenvolvimento de Interfaces Web da PUCMinas. Essa é a versão v1.0 composta pelo conteúdo em HTML, a formatação em CSS e o comportamento em Javascript com a utilização da biblioteca jQuery e uso do AJAX. A comunicação com o servidor é feita por meio de requisições http. A API utilizada para preencher os conteúdos do site é a The Movie DB, no intuito de praticar as metodologias aprendidas durante o semestre e obter proficiência no desenvolvimento e utilização de tecnologias empregadas na web. Informo que todas as imagens e vídeos utilizados na composição desse trabalho foram retirados de fontes de livre acesso na web. O site aqui apresentado não possui fins comerciais.
